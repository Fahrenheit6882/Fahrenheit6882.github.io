# Computer Aided Design (CAD)

## Onshape

[Onshape](https://onshape.com) is the CAD software we use to design our robot. This is fully online so all you need is a Web Browser. 

Follow [this guide](https://onshape4frc.com/getting-started) to sign up for an account. 

We have a Onshape Team named `Robotics Team 6882`. After you create a OnShape account, a Mentor can add you to the team.
